GitHub Repository Cheat Sheet
 1. Plain Git CLI
 mkdir -Force "C:\python\new_project"
 cd "C:\python\new_project"
 echo "# my-project" > README.md
 git init
 git add .
 git commit -m "first commit"
 git branch -M main
 git remote add origin https://github.com/amoorinet/repo.git
 git push -u origin main
 git pull origin main # later to update
 git fetch origin # check for updates without merging
 2. GitHub CLI (gh)
 gh auth login
 mkdir -Force "C:\python\new_project"
 cd "C:\python\new_project"
 echo "print('hello')" > app.py
 git init
 git add .
 git commit -m "commit all"
 git branch -M main
 gh repo create repo_name --public --source=. --remote=origin --push
 git pull # later updates
 git fetch # only check updates
 3. GitHub Desktop
 Open GitHub Desktop → File → New Repository
 Fill details (Name, Path, etc.), check Initialize with README
 Click Create Repository
 Click Publish Repository (choose public/private)
 Click Fetch origin to check updates
 Click Pull origin to bring updates
4. VS Code (Insiders/Stable)
 Open VS Code → File → Open Folder → select your folder
 Source Control tab → Initialize Repository
 Stage changes (+) → Write commit message → Commit
 Click Publish Branch (to push to GitHub)
 Click Pull to update from GitHub
 Command Palette → Git: Fetch (to only check updates
